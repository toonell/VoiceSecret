# 🎵 V Secret

**V Secret** is a professional **audio steganography tool** that allows you to:  
**🎤 Record voice, 🔐 embed secret messages in WAV audio files, and 💌 extract hidden messages.**  

It is designed for **ease of use**, compatible with most WAV recordings, and perfect for **secure communication or educational purposes**.

---

## ✨ Features

- **🎤 Voice Recording** – Record audio directly from your microphone.  
- **🔐 Embed Secret Messages** – Hide text messages in any WAV file.  
- **💌 Extract Hidden Messages** – Retrieve embedded messages from WAV files.  
- **🖥️ Terminal-Friendly** – Fully interactive and easy-to-use in the terminal.  
- **🔄 Universal Audio Support** – Works with WAV files recorded from any device.  
- **✅ Clean & Professional Interface** – Lightweight and visually clear output.

---

## ⚡ Usage

1. **Record or Use Existing Audio**  
   - Record a new audio clip or provide an existing WAV file.  

2. **Choose an Operation**  
   - **Embed** a secret message in the WAV file.  
   - **Extract** a secret message from a WAV file.  

3. **Follow Prompts**  
   - For **embedding**: enter the output filename and your secret message.  
   - For **extraction**: the tool will display the hidden message if present.
